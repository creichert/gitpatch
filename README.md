# What is gitpatch

gitpatch is a simple tool, you can use it to mail a git patch file with message-mode.


<a id="org60879a1"></a>

## Installation

1.  Config melpa source, please read: <http://melpa.org/#/getting-started>
2.  M-x package-install RET gitpatch RET


<a id="orga9d582e"></a>
